A local google keep clone, based on react

## Installation

Install google-keep-clone-2.0 with npm

```bash
  cd google-keep-clone-2.0
  npm start
```

## Run with Docker
```bash
  cd google-keep-clone-2.0
```

```bash
docker-compose up --build
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
